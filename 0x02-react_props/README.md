This React Starter Kit provides a foundation for building React applications while demonstrating fundamental concepts like creating components, reusing components, passing properties, defining types, using fragments, and incorporating Enzyme for testing React components.

## Getting Started

### Prerequisites

Ensure you have Node.js and npm installed on your machine.

- Node.js: [Download and Install Node.js](https://nodejs.org/)
- npm: Included with Node.js installation

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/wellingtonmwadali/alx-react
   ```

2. Navigate to the project directory:

   ```bash
   cd react-starter-kit
   ```

3. Install dependencies:

   ```bash
   npm install
   ```
